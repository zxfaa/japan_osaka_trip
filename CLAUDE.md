# 大阪・京都 2026 旅遊攻略網站

## 專案概述

這是一個為期 6 天 5 夜（2026/8/27 四 ～ 9/1 二）的日本關西旅遊攻略網站，  
供 **6 人團體** 使用，目前已部署在 Cloudflare Pages 上。

原始 HTML 檔案為單一 `osaka_kyoto_complete.html`，  
內含 4 個分頁（行程 / 交通 / 住宿 / 餐廳），以純 HTML + CSS + 原生 JS 建構，無框架依賴。

## 行程規劃（已確定）

### Day 1｜8/27（四）抵達 → 京都夜遊
- 06:40 桃園出發，10:25 抵達關西機場
- 搭 JR HARUKA 特急（80 分）直達京都車站
- 下午：錦市場閒逛（玉子燒、豆乳冰淇淋、串燒小吃）
- 晚餐：**鴨川納涼床**（5/1～9/30 季節限定，需提前預約）
- 夜遊：祇園花見小路 + 先斗町

### Day 2｜8/28（五）京都全天 · 東山路線
- 早：**伏見稻荷大社**（8:30 前抵達避人潮，JR 稻荷站）
- 午：清水坂周邊午餐（湯豆腐 / 懷石定食 / 茶寮都路里）
- 午後：**清水寺** → 二年坂 → 三年坂 → 八坂神社 → 祇園白川
- 晚：燒肉弘 或 祇園周邊居酒屋
- 夜（可選）：京都塔展望台（¥900）或車站大階梯 LED 燈光秀（免費）

### Day 3｜8/29（六）嵐山上午 → 下午移動大阪
- 早（8:00 前）：**嵐山**竹林 + 渡月橋（JR 嵯峨嵐山站）
- ⚠️ 週六巴士 11 號停駛，嵐山→金閣寺建議計程車（¥1,500–2,000，6 人分攤划算）
- 午：嵐山附近吃完移動
- 14:00：阪急電鐵 河原町 → 梅田（40 分）+ 地鐵至難波飯店
- 晚：道頓堀掃街（章魚燒、大阪燒、串炸）

### Day 4｜8/30（日）大阪 · 逛街體驗
- 上午：**黑門市場**（海鮮丼、烤扇貝、和牛串）
- 下午：**心齋橋** + 美國村 + 藥妝掃貨（松本清、大國藥妝、唐吉軻德）
- 傍晚（可選）：大阪城公園（外圍免費，天守閣 ¥600）
- 晚餐：**帝王蟹 / 螃蟹料理**（蟹道樂 或 FUJIYAMA TOKYO）

### Day 5｜8/31（一）大阪 · 梅田生活感
- 上午：天神橋筋商店街 + **大阪生活今昔館**
- 下午：**梅田**百貨（阪急、Grand Front、LUCUA）
- 傍晚：**梅田藍天大廈**展望台（15:00 前入場可免費，大阪周遊卡）
- 晚餐：**和牛燒肉**（板前燒肉一斗 / 燒肉五苑 / 播重壽喜燒）

### Day 6｜9/1（二）最後採買 → 回台
- 上午：心齋橋藥妝最後衝刺（551 蓬萊豬肉包必買）
- **⚠️ 最晚 11:45 從難波出發**（南海 Rapi:t 34 分到機場，班機 15:10）
- 15:10 IT211 KIX → 桃園

## 現有網站結構

```
osaka_kyoto_complete.html   ← 主檔案（單一 HTML，已部署 Cloudflare Pages）
CLAUDE.md                   ← 本檔案
```

### 網站架構
- **純 HTML + CSS + 原生 JavaScript**，無框架依賴
- **4 個分頁**，透過 `showTab()` 函式切換：
  - `#itinerary` — 行程（Day Card 展開式時間軸）
  - `#transport` — 交通資訊表格
  - `#hotels` — 住宿卡片
  - `#food` — 餐廳卡片
- **響應式設計**，max-width: 720px，支援手機瀏覽
- **Nav 為 sticky**，頂部固定分頁切換

### 主要 JS 函式

```js
showTab(id, btn)   // 切換分頁顯示
toggleDay(i)       // 展開/收合 Day Card
renderDays()       // 從 days[] 陣列渲染行程時間軸
renderTransport()  // 從 tData[] 陣列渲染交通表格
```

### 行程資料結構（days[]）

```js
const days = [
  {
    date: "8/27（四）",
    title: "抵達 → 京都夜遊",
    badge: "bk",        // bk=京都紫 / bo=大阪藍 / bt=移動日綠
    bl: "京都",         // badge 文字
    slots: [
      {
        t: "06:40",     // 時間欄
        h: "IT210 桃園出發",  // 標題
        d: "關西機場約 10:25 抵達。",  // 描述
        tags: ["推薦"]  // 可選，顯示小 tag
      }
    ],
    rest: {             // 可選，晚餐選項方塊
      title: "晚餐選擇",
      opts: ["選項1", "選項2"]
    },
    tip: "底部提示文字"  // 可選，紫色左邊框提示
  }
]
```

### 交通資料結構（tData[]）

```js
const tData = [
  {
    id: "t0",  // 對應 HTML 中的 <div id="t0">
    rows: [
      {
        f: "起點",
        t: "終點",
        d: "77.3 km",   // 距離
        m: "JR HARUKA", // 交通方式
        time: "80 分",
        type: "train",  // train / walk / bus（控制 tag 顏色）
        note: "備註"    // 可選
      }
    ]
  }
]
```

## 設計風格規範

### 主色調
| 用途 | 色值 |
|------|------|
| 主色（Header / 分頁底線） | `#534AB7`（品牌紫） |
| 背景 | `#f5f4f0` |
| 卡片背景 | `#ffffff` |
| 邊框 | `0.5px solid #e0ddd5` |

### Badge 顏色
| Class | 對應 | 背景 | 文字 |
|-------|------|------|------|
| `.bk` | 京都 | `#EEEDFE` | `#3C3489` |
| `.bo` | 大阪 | `#E6F1FB` | `#0C447C` |
| `.bt` | 移動日/返程 | `#EAF3DE` | `#3B6D11` |

### Slot Tag 顏色
| Class | 用途 | 背景 | 文字 |
|-------|------|------|------|
| `.stag-hot` | 必去/推薦 | `#FAECE7` | `#993C1D` |
| `.stag-food` | 吃/餐廳 | `#EAF3DE` | `#3B6D11` |
| `.stag-night` | 夜間景點 | `#EEEDFE` | `#3C3489` |
| `.stag-def` | 其他預設 | `#f0eeea` | `#666` |

### 交通 Tag 顏色
| Class | 用途 |
|-------|------|
| `.ttag-train` | 電車（藍） |
| `.ttag-walk` | 步行（綠） |
| `.ttag-bus` | 巴士（橘） |

### 圓角規則
- 大卡片（`.day-card`, `.info-card`）：`border-radius: 12px`
- 小元素（tag, badge, tip box）：`border-radius: 8–20px`
- Tip 提示塊：左邊框 `3px solid #7F77DD`，背景 `#EEEDFE`
